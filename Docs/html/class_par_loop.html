Detailed Description

Utility class for provinding a loop executing a callback function #cbf in a thread. After executing this function this thread will sleep for #loopTimeSeconds seconds. This class in non-copyable. RAII: the thread is created by the constructor and is stopped in the destructor # Parloop.
